Cha Virtual Machine
====================


INTRODUCTION
------------

ChaVM is a minimum implementation of Java SE 7(Java Virtual Machine Specification Java SE 7 Edition) which
can be find at the following link:
https://docs.oracle.com/javase/specs/jvms/se7/jvms7.pdf
